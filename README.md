# Contact Form Project (Full Stack)

A simple full-stack contact form built with HTML, CSS, JavaScript, and Node.js.

## Features
- User-friendly contact UI  
- Stores submitted messages in a JSON file  
- Backend API using Express  
- Clean folder structure

## Technologies
- HTML  
- CSS  
- JavaScript  
- Node.js  
- Express.js
